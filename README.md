# Online Shopping Management System (Java OOP)

A console-based Java application designed to simulate a real e-commerce platform. The system allows customers to browse products, manage their shopping cart, handle balances, apply dynamic discounts, and generate detailed invoices.

## 🚀 OOP Concepts Implemented
- **Abstraction:** Implemented using an abstract `Customer` class with abstract method `calculateDiscount()`.
- **Inheritance:** `RegularCustomer` and `PremiumCustomer` inherit common properties from the base `Customer` class.
- **Encapsulation:** All class fields are kept `private` and accessed securely via public getters and setters.
- **Polymorphism:** Method overriding is used to apply different discount logic for each customer type.

## 🛠️ Features Included
1. Add Product & Display Inventory
2. Search Product & Manage Stock Quantity
3. Shopping Cart Operations (Add, Remove, View, Calculate Total)
4. Order Processing & Invoice Generation
5. Balance Management (Add Balance, View Balance)
6. Dynamic Discount Calculations (Premium vs. Regular Customers)

## 💻 Tech Stack
- **Language:** Java (JDK 11 or higher)
- **Concepts:** Object-Oriented Programming (OOP), Java Collections (ArrayList), Exception Handling.
